# README for Medi-nova
## Demo Video
- https://drive.google.com/file/d/1WkygLzSQfD_LJ7C2aLTigbshrABZ6Luk/view
  
This project is a comprehensive web and mobile application designed to predict diseases using multiple AI models. The application leverages IoT devices to collect essential health data and employs advanced machine learning algorithms to analyze this data, providing proactive health insights and predictions.

## Features

- **Multi-Modal AI Models:** Utilizes various AI models to predict diseases based on real-time data.
- **IoT Integration:** Collects vital health data such as BP, SpO2, and BPM using IoT devices.
- **Real-Time Data Processing:** Stores collected data in a database and uses it to make accurate disease predictions.
- **Predictive Analysis:** Provides disease predictions based on current and historical data.
- **Medical Chatbot:** Offers a chatbot for interactive medical advice and support.
- **Weather-Based Disease Prediction:** Analyzes weather conditions to predict potential health risks.
- **First Aid Operations:** Guides users through first aid procedures.
- **Mind Calm Exercise Center:** Provides exercises for mental wellness.
- **Lab Report Summarizer:** Summarizes lab reports for easy understanding.
- **Nearby Hospital Locator:** Helps users find nearby hospitals quickly.
- **Personalized Health Management:** Offers tailored health management plans for individuals.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/medical-ai-app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd medical-ai-app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
   
## Usage

- **IoT Device Setup:** Ensure IoT devices are connected and configured to capture health data.
- **Data Collection:** Allow the app to collect data from IoT devices.
- **Disease Prediction:** Access real-time predictions on the dashboard.
- **Chatbot Interaction:** Use the medical chatbot for health queries.
- **Explore Features:** Utilize additional features like first aid guidance, mind exercises, and hospital locator.
